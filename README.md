## Packages Used

- expo: Expo development environment used instead of React Native CLI
- typescript: typescript instead of javascript is used, which ensures type safety and better data flow
- react navigation: Blank typescript template is used for the expo project along react navigation for routing
- nativewind and tailwindcss: used instead of Stylesheet.create
- redux toolkit: Used for central state management and RTQ Query
- expo-image-picker: Used for accessing camera and gallery media
- expo-dev-client: used for building and testing app on actual device
